File.File {#File.File}
=========

> [File]{role="ref"} **File** ([string]{role="ref"} **path**)

Parameters
----------

  ---------- --------------------------------------------------------------
  **path**   The full or partial path name of the file, in
             platform-specific or URI format.
  ---------- --------------------------------------------------------------

Description
-----------

Creates and returns a new File object referring to a given file system
location.
