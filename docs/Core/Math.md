Math {#Math}
====

Description
-----------

A global object containing a set of math functions and constants.

### Static Properties

  ----------------------------------------- -------------------------------------------
  [E\<Math.E\>]{role="ref"} readonly        Euler\'s constant and the base of natural
                                            logarithms.

  [LN10\<Math.LN10\>]{role="ref"} readonly  The natural logarithm of 10.

  [LN2\<Math.LN2\>]{role="ref"} readonly    The natural logarithm of 2.

  [LOG10E\<Math.LOG10E\>]{role="ref"}       The base 10 logarithm of e.
  readonly                                  

  [LOG2E\<Math.LOG2E\>]{role="ref"}         The base 2 logarithm of e.
  readonly                                  

  [PI\<Math.PI\>]{role="ref"} readonly      The ratio of the circumference of a circle
                                            to its diameter.

  [SQRT1\_2\<Math.SQRT1\_2\>]{role="ref"}   The reciprocal of the square root of 1/2.
  readonly                                  

  [SQRT2\<Math.SQRT2\>]{role="ref"}         The square root of 2.
  readonly                                  
  ----------------------------------------- -------------------------------------------

### Static Methods

  ------------------------------------- ----------------------------------------------
  [abs\<Math.abs\>]{role="ref"}         Returns the absolute value of a number.
  readonly                              

  [acos\<Math.acos\>]{role="ref"}       Returns the arc cosine(in radians) of a
  readonly                              number.

  [asin\<Math.asin\>]{role="ref"}       Returns the arc sin(in radians) of a number.
  readonly                              

  [atan\<Math.atan\>]{role="ref"}       Returns the arc tangent(in radians) of a
  readonly                              number.

  [atan2\<Math.atan2\>]{role="ref"}     Returns the arc tangent of the quotient of its
  readonly                              arguments (y/x).

  [ceil\<Math.ceil\>]{role="ref"}       Rounds the number up to the nearest integer.
  readonly                              

  [cos\<Math.cos\>]{role="ref"}         Returns the cosine of an angle provided in
  readonly                              radians.

  [exp\<Math.exp\>]{role="ref"}         Returns Math.E raised to the power of a
  readonly                              number.

  [floor\<Math.floor\>]{role="ref"}     Rounds a number down to the nearest integer.
  readonly                              

  [log\<Math.log\>]{role="ref"}         Returns the natural logarithm of a number.
  readonly                              

  [max\<Math.max\>]{role="ref"}         Returns the largest of zero or more numbers.
  readonly                              

  [min\<Math.min\>]{role="ref"}         Returns the smallest of zero or more numbers.
  readonly                              

  [pow\<Math.pow\>]{role="ref"}         Returns x raised to the power of y.
  readonly                              

  [random\<Math.random\>]{role="ref"}   Returns a pseudo-random number from 0.0 up to
  readonly                              but not including 1.0.

  [round\<Math.round\>]{role="ref"}     Rounds a number to the nearest integer.
  readonly                              

  [sin\<Math.sin\>]{role="ref"}         Returns the sine of an angle provided in
  readonly                              radians.

  [sqrt\<Math.sqrt\>]{role="ref"}       Returns the square root of a number.
  readonly                              

  [tan\<Math.tan\>]{role="ref"}         Returns the tangent of an angle provided in
  readonly                              radians.
  ------------------------------------- ----------------------------------------------
