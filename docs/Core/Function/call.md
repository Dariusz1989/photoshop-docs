Function.call {#Function.call}
=============

> any **call** ([Object]{role="ref"} **thisObj**, any **argument**)

Parameters
----------

  -------------- --
  **thisObj**    

  **argument**   
  -------------- --

Description
-----------

Apply a?this?object and arguments to a function.

This function is different from?apply(); here, the arguments are
supplied one by one.
