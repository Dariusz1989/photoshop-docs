Math.atan2 {#Math.atan2}
==========

> [number]{role="ref"} **atan2** ([number]{role="ref"} **y**,
> [number]{role="ref"} **x**)

Parameters
----------

  ------- -----------
  **y**   A number.

  **x**   A number.
  ------- -----------

Description
-----------

Returns the arc tangent of the quotient of its arguments (y/x).
