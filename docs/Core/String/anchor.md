String.anchor {#String.anchor}
=============

> [string]{role="ref"} **anchor** ([string]{role="ref"} **name**)

Parameters
----------

  ---------- -------------------------------------------------------
  **name**   The text to be stored in the anchors\' name attribute.
  ---------- -------------------------------------------------------

Description
-----------

Returns a string consisting of this string enclosed in a \<a\> tag.
