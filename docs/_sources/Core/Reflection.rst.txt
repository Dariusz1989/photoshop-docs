.. _Reflection:

================================================
Reflection
================================================


Description
-----------

Provides information about a class.




Properties
^^^^^^^^^^

+---------------------------------------------------------------+---------------------------------------------+
| :ref:`description<Reflection.description>` readonly           | The long description text.                  |
+---------------------------------------------------------------+---------------------------------------------+
| :ref:`help<Reflection.help>` readonly                         | The short description text.                 |
+---------------------------------------------------------------+---------------------------------------------+
| :ref:`methods<Reflection.methods>` readonly                   | An array of method descriptions.            |
+---------------------------------------------------------------+---------------------------------------------+
| :ref:`name<Reflection.name>` readonly                         | The class name.                             |
+---------------------------------------------------------------+---------------------------------------------+
| :ref:`properties<Reflection.properties>` readonly             | An array of property descriptions.          |
+---------------------------------------------------------------+---------------------------------------------+
| :ref:`sampleCode<Reflection.sampleCode>` readonly             | Sample code, if present.                    |
+---------------------------------------------------------------+---------------------------------------------+
| :ref:`sampleFile<Reflection.sampleFile>` readonly             | A file containing sample code. May be null. |
+---------------------------------------------------------------+---------------------------------------------+
| :ref:`staticMethods<Reflection.staticMethods>` readonly       | An array of class method descriptions.      |
+---------------------------------------------------------------+---------------------------------------------+
| :ref:`staticProperties<Reflection.staticProperties>` readonly | An array of class property descriptions.    |
+---------------------------------------------------------------+---------------------------------------------+







Methods
^^^^^^^

+-----------------------------------------+------------------------------------------------------+
| :ref:`find<Reflection.find>` readonly   | Finds an element description by name.                |
+-----------------------------------------+------------------------------------------------------+
| :ref:`toXML<Reflection.toXML>` readonly | Returns this class information as XML in OMV format. |
+-----------------------------------------+------------------------------------------------------+
