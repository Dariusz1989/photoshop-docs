.. _NoiseDistribution:

================================================
NoiseDistribution
================================================


Description
-----------



The distribution method to use when applying an Add Noise filter.




Static Properties
^^^^^^^^^^^^^^^^^

+------------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------+
| :ref:`GAUSSIAN<NoiseDistribution.GAUSSIAN>` readonly | Distributes color values of noise along a bell-shaped curve, creating a speckled effect.                                          |
+------------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------+
| :ref:`UNIFORM<NoiseDistribution.UNIFORM>` readonly   | Distributes color values of noise using random numbers between 0 and plus or minus the specified value, creating a subtle effect. |
+------------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------+
