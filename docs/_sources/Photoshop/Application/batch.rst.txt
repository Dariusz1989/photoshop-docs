================================================
Application.batch
================================================

   :ref:`string` **batch** (:ref:`AliasArray` **inputFiles**, :ref:`string` **action**, :ref:`string` **from**, :ref:`BatchOptions` **options**)


Parameters
----------

+----------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **inputFiles** | The files to operate on.                                                                                                                                        |
+----------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **action**     | The name of the action to play (note that the Action name is case-sensitive and must match the name in the Actions palette).                                    |
+----------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **from**       | The name of the action set containing the action being played (note that the Action Set name is case-sensitive and must match the name in the Actions palette). |
+----------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **options**    | Options for batch automation.                                                                                                                                   |
+----------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------+



Description
-----------

Runs the batch automation routine; analogous to using the Batch command in Photoshop.
