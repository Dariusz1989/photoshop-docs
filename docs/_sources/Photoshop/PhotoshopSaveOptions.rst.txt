================================================
PhotoshopSaveOptions
================================================


Description
-----------

Options for saving a Photoshop document.






Static Properties
^^^^^^^^^^^^^^^^^

+---------------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`alphaChannels<PhotoshopSaveOptions.alphaChannels>` readonly         | If true, the alpha channels are saved.                  |
+---------------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`annotations<PhotoshopSaveOptions.annotations>` readonly             | If true, the annotations are saved.                     |
+---------------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`embedColorProfile<PhotoshopSaveOptions.embedColorProfile>` readonly | If true, the color profile is embedded in the document. |
+---------------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`layers<PhotoshopSaveOptions.layers>` readonly                       | If true, the layers are saved.                          |
+---------------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`spotColors<PhotoshopSaveOptions.spotColors>` readonly               | If true, spot colors are saved.                         |
+---------------------------------------------------------------------------+---------------------------------------------------------+
