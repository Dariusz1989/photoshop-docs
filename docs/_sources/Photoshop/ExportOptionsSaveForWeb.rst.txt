.. _ExportOptionsSaveForWeb:

================================================
ExportOptionsSaveForWeb
================================================


Description
-----------

Options for exporting Save For Web files.






Static Properties
^^^^^^^^^^^^^^^^^

+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`PNG8<ExportOptionsSaveForWeb.PNG8>` readonly                             | If true, uses 8 bits. If false, uses 24 bits. Valid only when 'format' = PNG.                            |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`blur<ExportOptionsSaveForWeb.blur>` readonly                             | The amount of blur to apply to the image to reduce artifacts.                                            |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`colorReduction<ExportOptionsSaveForWeb.colorReduction>` readonly         | The color reduction algorithm.                                                                           |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`colors<ExportOptionsSaveForWeb.colors>` readonly                         | The number of colors in the palette.                                                                     |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`dither<ExportOptionsSaveForWeb.dither>` readonly                         | The type of dither.                                                                                      |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`ditherAmount<ExportOptionsSaveForWeb.ditherAmount>` readonly             | The amount of dither. Valid only when 'dither' = diffusion.                                              |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`format<ExportOptionsSaveForWeb.format>` readonly                         | The file format to use. Save For Web supports only Compuserve GIF, JPEG, PNG-8, PNG-24, and BMP formats. |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`includeProfile<ExportOptionsSaveForWeb.includeProfile>` readonly         | If true, includes the document's embedded profile.                                                       |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`interlaced<ExportOptionsSaveForWeb.interlaced>` readonly                 | If true, the image downloads in multiple passes, progressive.                                            |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`lossy<ExportOptionsSaveForWeb.lossy>` readonly                           | The amount of lossiness allowed.                                                                         |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`matteColor<ExportOptionsSaveForWeb.matteColor>` readonly                 | The colors to blend transparent pixels against.                                                          |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`optimized<ExportOptionsSaveForWeb.optimized>` readonly                   | If true, creates smaller but less compatible files.                                                      |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`quality<ExportOptionsSaveForWeb.quality>` readonly                       | The quality of the produced image (as a percentage). Range: 0 to 100.                                    |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`transparency<ExportOptionsSaveForWeb.transparency>` readonly             | If true, transparent areas of the image are included in the saved image.                                 |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`transparencyAmount<ExportOptionsSaveForWeb.transparencyAmount>` readonly | The amount of transparency dither. Valid only when 'transparency' = true.                                |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`transparencyDither<ExportOptionsSaveForWeb.transparencyDither>` readonly | The transparency dither algorithm.                                                                       |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| :ref:`webSnap<ExportOptionsSaveForWeb.webSnap>` readonly                       | The tolerance amount within which to snap close colors to web palette colors.                            |
+--------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------+
