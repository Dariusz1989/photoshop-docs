.. _PICTResourceSaveOptions:

================================================
PICTResourceSaveOptions
================================================


Description
-----------

Options for saving a document as a PICT resource file.






Static Properties
^^^^^^^^^^^^^^^^^

+------------------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`alphaChannels<PICTResourceSaveOptions.alphaChannels>` readonly         | If true, the alpha channels are saved.                  |
+------------------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`compression<PICTResourceSaveOptions.compression>` readonly             | The compression type.                                   |
+------------------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`embedColorProfile<PICTResourceSaveOptions.embedColorProfile>` readonly | If true, the color profile is embedded in the document. |
+------------------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`name<PICTResourceSaveOptions.name>` readonly                           | The name of PICT resource.                              |
+------------------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`resolution<PICTResourceSaveOptions.resolution>` readonly               | The number of bits per pixel.                           |
+------------------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`resourceID<PICTResourceSaveOptions.resourceID>` readonly               | The ID of the PICT resource.                            |
+------------------------------------------------------------------------------+---------------------------------------------------------+
