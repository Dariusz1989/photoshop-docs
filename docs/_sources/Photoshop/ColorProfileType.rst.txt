================================================
ColorProfileType
================================================


Description
-----------



The type of color model that defines the document's working space.




Static Properties
^^^^^^^^^^^^^^^^^

+---------------------------------------------------+--------------------------------------------------------------+
| :ref:`CUSTOM<ColorProfileType.CUSTOM>` readonly   | Color manages this document using a custom color profile.    |
+---------------------------------------------------+--------------------------------------------------------------+
| :ref:`NONE<ColorProfileType.NONE>` readonly       | The document is not color managed.                           |
+---------------------------------------------------+--------------------------------------------------------------+
| :ref:`WORKING<ColorProfileType.WORKING>` readonly | Color manages this document using the working color profile. |
+---------------------------------------------------+--------------------------------------------------------------+
