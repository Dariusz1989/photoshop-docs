.. _PaintingCursors:

================================================
PaintingCursors
================================================


Description
-----------



The pointer for the following tools: Marquee, Lasso, Polygonal Lasso, Magic Wand, Crop, Slice, Patch Eyedropper, Pen, Gradient, Line, Paint Bucket, Magnetic Lasso, Magnetic Pen, Freeform Pen, Measure, and Color Sampler.




Static Properties
^^^^^^^^^^^^^^^^^

+------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| :ref:`BRUSHSIZE<PaintingCursors.BRUSHSIZE>` readonly | Displays cursors as brush shapes representing the size of the current brush. Valid only for painting cursors. |
+------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| :ref:`PRECISE<PaintingCursors.PRECISE>` readonly     | Displays pointers as cross hairs.                                                                             |
+------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| :ref:`STANDARD<PaintingCursors.STANDARD>` readonly   | Displays pointers as tool icons.                                                                              |
+------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
