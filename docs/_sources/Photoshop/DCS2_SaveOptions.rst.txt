.. _DCS2_SaveOptions:

================================================
DCS2_SaveOptions
================================================


Description
-----------

Options for saving a document in Photoshop DCS 2.0 format.






Static Properties
^^^^^^^^^^^^^^^^^

+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`DCS<DCS2_SaveOptions.DCS>` readonly                             | The DCS type.                                                                                               |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`embedColorProfile<DCS2_SaveOptions.embedColorProfile>` readonly | If true, the color profile is embedded in the document.                                                     |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`encoding<DCS2_SaveOptions.encoding>` readonly                   | The type of encoding to use for document.                                                                   |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`halftoneScreen<DCS2_SaveOptions.halftoneScreen>` readonly       | If true, includes halftone screen.                                                                          |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`interpolation<DCS2_SaveOptions.interpolation>` readonly         | If true, image interpolation is used.                                                                       |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`multiFileDCS<DCS2_SaveOptions.multiFileDCS>` readonly           | If true, saves color channels as multiple files.                                                            |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`preview<DCS2_SaveOptions.preview>` readonly                     | The type of preview.                                                                                        |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`spotColors<DCS2_SaveOptions.spotColors>` readonly               | If true, the spot colors are saved.                                                                         |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`transferFunction<DCS2_SaveOptions.transferFunction>` readonly   | If true, includes transfer functions in the document to compensate for dot gain between the image and film. |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`vectorData<DCS2_SaveOptions.vectorData>` readonly               | If true, includes vector data. Valid only if the document contains vector data (un-rasterized text).        |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
