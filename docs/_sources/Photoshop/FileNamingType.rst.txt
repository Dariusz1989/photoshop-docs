================================================
FileNamingType
================================================


Description
-----------



File naming options for the batch command.




Static Properties
^^^^^^^^^^^^^^^^^

+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`DDMM<FileNamingType.DDMM>` readonly                           | Uses the date formatted as ddmm.                   |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`DDMMYY<FileNamingType.DDMMYY>` readonly                       | Uses the date formatted as ddmmyy.                 |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`DOCUMENTNAMELOWER<FileNamingType.DOCUMENTNAMELOWER>` readonly | Use the document name in lower case.               |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`DOCUMENTNAMEMIXED<FileNamingType.DOCUMENTNAMEMIXED>` readonly | Use the document name in mixed case.               |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`DOCUMENTNAMEUPPER<FileNamingType.DOCUMENTNAMEUPPER>` readonly | Use the document name in UPPER case.               |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`EXTENSIONLOWER<FileNamingType.EXTENSIONLOWER>` readonly       | Use the extension of the file in lower case.       |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`EXTENSIONUPPER<FileNamingType.EXTENSIONUPPER>` readonly       | Use the extension of the file in UPPER case.       |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`MMDD<FileNamingType.MMDD>` readonly                           | Uses the date formatted as mmdd.                   |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`MMDDYY<FileNamingType.MMDDYY>` readonly                       | Uses the date formatted as mmddyy.                 |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`SERIALLETTERLOWER<FileNamingType.SERIALLETTERLOWER>` readonly | Use letter serial number lower case (a, b, c, ...) |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`SERIALLETTERUPPER<FileNamingType.SERIALLETTERUPPER>` readonly | Use letter serial number UPPER case (A, B, C, ...) |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`SERIALNUMBER1<FileNamingType.SERIALNUMBER1>` readonly         | Use a 1-digit serial number.                       |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`SERIALNUMBER2<FileNamingType.SERIALNUMBER2>` readonly         | Use a 2-digit serial number.                       |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`SERIALNUMBER3<FileNamingType.SERIALNUMBER3>` readonly         | Use a 3-digit serial number.                       |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`SERIALNUMBER4<FileNamingType.SERIALNUMBER4>` readonly         | Use a 4-digit serial number.                       |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`YYDDMM<FileNamingType.YYDDMM>` readonly                       | Uses the date formatted as yyddmm.                 |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`YYMMDD<FileNamingType.YYMMDD>` readonly                       | Uses the date formatted as yymmdd.                 |
+---------------------------------------------------------------------+----------------------------------------------------+
| :ref:`YYYYMMDD<FileNamingType.YYYYMMDD>` readonly                   | Uses the date formatted as yyyymmdd.               |
+---------------------------------------------------------------------+----------------------------------------------------+
