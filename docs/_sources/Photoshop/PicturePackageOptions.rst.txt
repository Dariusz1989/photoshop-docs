================================================
PicturePackageOptions
================================================


Description
-----------

Options that can be specified for a Picture Package.






Static Properties
^^^^^^^^^^^^^^^^^

+------------------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`content<PicturePackageOptions.content>` readonly           | The content information.                                              |
+------------------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`flatten<PicturePackageOptions.flatten>` readonly           | If true, flattens all layers in the final document.                   |
+------------------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`font<PicturePackageOptions.font>` readonly                 | The font used for security text.                                      |
+------------------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`fontSize<PicturePackageOptions.fontSize>` readonly         | The font size.                                                        |
+------------------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`layout<PicturePackageOptions.layout>` readonly             | The layout to use to generate the picture package.                    |
+------------------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`mode<PicturePackageOptions.mode>` readonly                 | The document color mode.                                              |
+------------------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`opacity<PicturePackageOptions.opacity>` readonly           | The web page security text opacity (as a percentage) Range: 0 to 100. |
+------------------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`resolution<PicturePackageOptions.resolution>` readonly     | The resolution of the document (in pixels per inch)                   |
+------------------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`text<PicturePackageOptions.text>` readonly                 | The picture package custom text.                                      |
+------------------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`textColor<PicturePackageOptions.textColor>` readonly       | The color of the security text.                                       |
+------------------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`textPosition<PicturePackageOptions.textPosition>` readonly | The position of the security text.                                    |
+------------------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`textRotate<PicturePackageOptions.textRotate>` readonly     | The orientation of the security text.                                 |
+------------------------------------------------------------------+-----------------------------------------------------------------------+
