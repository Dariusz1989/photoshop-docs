.. _TextureType:

================================================
TextureType
================================================


Description
-----------



The type of texture or glass surface image to load for a texturizer or glass filter.




Static Properties
^^^^^^^^^^^^^^^^^

+------------------------------------------------+-----------------------------------------------------------------+
| :ref:`BLOCKS<TextureType.BLOCKS>` readonly     | The image appears as if viewed through glass blocks.            |
+------------------------------------------------+-----------------------------------------------------------------+
| :ref:`CANVAS<TextureType.CANVAS>` readonly     | The image appears as if painted on canvas.                      |
+------------------------------------------------+-----------------------------------------------------------------+
| :ref:`FILE<TextureType.FILE>` readonly         | Texture from an existing document.                              |
+------------------------------------------------+-----------------------------------------------------------------+
| :ref:`FROSTED<TextureType.FROSTED>` readonly   | The image appears as if frosted.                                |
+------------------------------------------------+-----------------------------------------------------------------+
| :ref:`TINYLENS<TextureType.TINYLENS>` readonly | The image appears as if viewed through an array of tiny lenses. |
+------------------------------------------------+-----------------------------------------------------------------+
