================================================
ScriptUIGraphics.moveTo
================================================

   :ref:`Point` **moveTo** (:ref:`Number` **x**, :ref:`Number` **y**)


Parameters
----------

+-------+-----------------------------------------------------------------------------+
| **x** | The X coordinate for the new point, relative to the origin of this element. |
+-------+-----------------------------------------------------------------------------+
| **y** | The Y coordinate for the new point, relative to the origin of this element. |
+-------+-----------------------------------------------------------------------------+



Description
-----------

Adds a given point to the?currentPath, and makes it the current drawing position.

Returns the?Point?object which is the new value of?currentPoint.
