ScriptUIBrush {#ScriptUIBrush}
=============

Description
-----------

A painting brush that encapsulates a color or pattern used to fill
paths.

Create with?ScriptUIGraphics.newBrush(). Use as a value of
?backgroundColor?properties, and pass as an argument to
the?fillPath()?method.

### Properties

  -------------------------------------------- ------------------------------
  [color\<ScriptUIBrush.color\>]{role="ref"}   The brush color.
  readonly                                     

  [theme\<ScriptUIBrush.theme\>]{role="ref"}   The theme name.
  readonly                                     

  [type\<ScriptUIBrush.type\>]{role="ref"}     The brush type, solid or
  readonly                                     theme.
  -------------------------------------------- ------------------------------
