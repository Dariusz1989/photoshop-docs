DropDownList.remove {#DropDownList.remove}
===================

> void **remove** (any **what**)

Parameters
----------

  ---------- ---------------------------------------------------------------
  **what**   The item or child to remove, specified by 0-based index, text
             value, or as a?ListItem?object.
  ---------- ---------------------------------------------------------------

Description
-----------

Removes a child item from the list.
