PICTCompression {#PICTCompression}
===============

Description
-----------

The compression type to use whan saving as PICT.

  readonly                                                           

  [JPEGLOWPICT\<PICTCompression.JPEGLOWPICT\>]{role="ref"} readonly  Low quality JPEG encoding (high

  readonly                                                           (very little compression).
