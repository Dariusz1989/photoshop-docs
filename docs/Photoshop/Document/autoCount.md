Document.autoCount {#Document.autoCount}

> void **autoCount** ([Channel]{role="ref"} **channel**, int
> **threshold**)

Parameters
----------

  --------------- -------------------------------------------------
  **channel**     The channel to use for counting.

  **threshold**   Threshold to use for counting. Range: 0 to 255.
