RegExp¶

Description¶

Wraps a regular expression.

Static Properties¶

$1 readonly The matched subexpression #1.
$2 readonly The matched subexpression #2.
$3 readonly The matched subexpression #3.
$4 readonly The matched subexpression #4.
$5 readonly The matched subexpression #5.
$6 readonly The matched subexpression #6.
$7 readonly The matched subexpression #7.
$8 readonly The matched subexpression #8.
$9 readonly The matched subexpression #9.
global Indicates whether the match is a global match.
ignoreCase Indicates whether the match is not case sensitive.
input The original input string.
lastMatch readonly The last match.
lastParen readonly The value of the last matched subexpression.
leftContext readonly The string before the match.
multiline Indicates whether the match matches multiple lines.
rightContext readonly The string after the match.

Constructors¶

RegExp Creates and returns a new RegExp object set to the value of the argument converted to a regular expression.

Methods¶

compile Compiles a string to a regular expression. Returns true if the compilation was successful.
exec Execute a regular expression.
test Execute a regular expression, and return true if there is a match.
toString Converts this RegExp object to a string.
