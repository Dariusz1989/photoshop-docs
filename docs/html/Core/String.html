String¶

Description¶

A character string. Each character is adressable by index.

Properties¶

length readonly The length of the string.

Constructors¶

String Returns a string representation of the value given as an argument.

Methods¶

anchor Returns a string consisting of this string enclosed in a <a> tag.
big Returns a string consisting of this string enclosed in a <big> tag.
blink Returns a string consisting of this string enclosed in a <blink> tag.
bold Returns a string consisting of this string enclosed in a <b> tag.
charAt Returns the character at the specified index.
charCodeAt Returns the Unicode value of the character at the given index.
concat If necessary, converts the one or more given values to strings.
fixed Returns a string consisting of this string enclosed in a <tt> tag.
fontcolor Returns a string consisting of this string enclosed in a <font> tag.
fontsize Returns a string consisting of this string enclosed in a <font> tag.
indexOf Returns the index within the string of the first occurrence of the specified string, starting the search at fromIndex if provided.
italics Returns a string consisting of this string enclosed in a <i> tag.
lastIndexOf Returns the index within the string of the last occurrence of the specified value.
link Returns a string consisting of this string enclosed in a <a> tag.
localeCompare Performs a localized comparison of two strings.
match Matches a string against a regular expression.
replace  
search  
slice Extracts a substring of the given string and returns it as a new string.
small Returns a string consisting of this string enclosed in a <small> tag.
split Splits a string into a group of substrings, places those strings in an array, and returns the array.
strike Returns a string consisting of this string enclosed in a <strike> tag.
sub Returns a string consisting of this string enclosed in a <sub> tag.
substr Returns a string containing the characters beginning at the specified index, start, through the specified number of characters.
substring Returns a substring of the given string by extracting characters from indexA up to but not including indexB.
sup Returns a string consisting of this string enclosed in a <sup> tag.
toLocaleLowerCase Returns a new string which contains all the characters of the original string converted to lowercase (localized).
toLocaleUpperCase Returns a new string which contains all the characters of the original string converted to uppercase (localized).
toLowerCase Returns a new string which contains all the characters of the original string converted to lowercase.
toSource Creates a string representation of this object that can be fed back to eval() to re-create an object. Works only with built-in classes.
toString Returns itself.
toUpperCase Returns a new string which contains all the characters of the original string converted to uppercase.
valueOf Returns itself.

Static Methods¶

fromCharCode Returns a string created by concatenation one or more characters specified as ASCII values.
