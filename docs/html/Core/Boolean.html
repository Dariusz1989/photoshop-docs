Boolean¶

Description¶

Wraps a Boolean value.

Constructors¶

Boolean Creates and returns a new Boolean object set to the value of the argument converted to a boolean.

Methods¶

toSource Creates a string representation of this object that can be fed back to eval() to re-create an object. Works only with built-in classes.
toString Returns the string representation of the value of bool.
valueOf Returns the primitive value of bool.
