Socket¶

Description¶

Creates a TCP/IP connection, or establishes a TCP/IP server.

Properties¶

connected readonly When true, the connection is active.
encoding Sets or retrieves the name of the encoding used to transmit data.
eof readonly When true, the receive buffer is empty.
error A message describing the most recent error. Setting this value clears any error message.
host readonly The name of the remote computer when a connection is established.
timeout The timeout in seconds to be applied to read or write operations.

Constructors¶

Socket Creates a new Socket object.

Methods¶

close Terminates the open connection.
listen Instructs the object to start listening for an incoming connection.
open Opens the connection for subsequent read/write operations.
poll Checks a listening object for a new incoming connection.
read Reads up to the specified number of characters from the connection. CR characters are ignored unless the encoding is set to “BINARY”.
readln Reads one line of text up to the next line feed.
write Concatenates all arguments into a single string and writes that string to the connection.
writeln Concatenates all arguments into a single string, appends a LF character, and writes that string to the connection.
