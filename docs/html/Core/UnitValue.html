UnitValue¶

Description¶

Represents a measurement as a combination of values and unit.

Note that this object is not available in all applications.

Properties¶

baseUnit readonly The base unit.
type readonly The unit name.
value readonly The numeric value.

Static Properties¶

baseUnit readonly The base unit for all conversions.

Methods¶

as readonly Returns this instance as a different unit.
convert readonly Converts this instance to a different unit.
