Function¶

Description¶

Wraps a built-in or JavaScript function.

Properties¶

arguments The function arguments, packed into an array.
arity readonly The number of formal arguments.
length readonly The number of formal arguments.
name readonly The function name.

Constructors¶

Function  

Methods¶

apply Apply a this object and an argument list to a function.
call Apply a this object and arguments to a function.
toSource Creates a string representation of this object that can be fed back to eval() to re-create an object. Works only with JavaScript functions.
toString Returns the function definition as a string.
