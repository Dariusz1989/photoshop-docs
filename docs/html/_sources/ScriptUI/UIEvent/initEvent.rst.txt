   void **initEvent** (:ref:`String` **type**, :ref:`Boolean` **captures**, :ref:`Boolean` **bubbles**, :ref:`Boolean` **cancelable**)


Parameters
----------

+----------------+--------------------------------------------------+
| **type**       | The event type.                                  |
+----------------+--------------------------------------------------+
| **captures**   | Set to true if this event can be captured.       |
+----------------+--------------------------------------------------+
