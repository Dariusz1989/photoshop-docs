================================================
Bounds
================================================


Description
-----------

Defines the boundaries of a window within the screen's coordinate space, or of a UI element within the container's coordinate space.

A Bounds object is created when you set an element's bounds property. You can set the property using a JavaScript object with properties named  left, top, right, bottom or x, y, width, height, or an array with 4 values in the order [x, y, wd, ht].


Properties
^^^^^^^^^^

+---------------------------------------+----------------------------------------------------------------------------------------------+
| :ref:`bottom<Bounds.bottom>`          | The vertical coordinate, a pixel offset from the origin of the element's coordinate space.   |
+---------------------------------------+----------------------------------------------------------------------------------------------+
| :ref:`height<Bounds.height>`          | The height in pixels.                                                                        |
+---------------------------------------+----------------------------------------------------------------------------------------------+
| :ref:`left<Bounds.left>`              | The horizontal coordinate, a pixel offset from the origin of the element's coordinate space. |
+---------------------------------------+----------------------------------------------------------------------------------------------+
| :ref:`length<Bounds.length>` readonly | The array length.                                                                            |
+---------------------------------------+----------------------------------------------------------------------------------------------+
| :ref:`right<Bounds.right>`            | The width in pixels.                                                                         |
+---------------------------------------+----------------------------------------------------------------------------------------------+
| :ref:`top<Bounds.top>`                | The height in pixels.                                                                        |
+---------------------------------------+----------------------------------------------------------------------------------------------+
| :ref:`width<Bounds.width>`            | The width in pixels.                                                                         |
+---------------------------------------+----------------------------------------------------------------------------------------------+
| :ref:`x<Bounds.x>`                    | The horizontal coordinate, a pixel offset from the origin of the element's coordinate space. |
+---------------------------------------+----------------------------------------------------------------------------------------------+
| :ref:`y<Bounds.y>`                    | The vertical coordinate, a pixel offset from the origin of the element's coordinate space.   |
+---------------------------------------+----------------------------------------------------------------------------------------------+
