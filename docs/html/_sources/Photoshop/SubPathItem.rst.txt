.. _SubPathItem:

================================================
SubPathItem
================================================


Description
-----------

Information about a path. You do not use the sub path item object to create a path. Rather, you create path segments using the sub path info object. Use the sub path item object to retrieve information about a path. All properties are read-only.




Properties
^^^^^^^^^^

+----------------------------------------------------+---------------------------------------------+
| :ref:`closed<SubPathItem.closed>` readonly         | If true, the path is closed.                |
+----------------------------------------------------+---------------------------------------------+
| :ref:`operation<SubPathItem.operation>` readonly   | The sub path operation on other sub paths.  |
+----------------------------------------------------+---------------------------------------------+
| :ref:`parent<SubPathItem.parent>` readonly         | The object's container.                     |
+----------------------------------------------------+---------------------------------------------+
| :ref:`pathPoints<SubPathItem.pathPoints>` readonly | The path points collection in the sub path. |
+----------------------------------------------------+---------------------------------------------+
| :ref:`typename<SubPathItem.typename>` readonly     | The class name of the object.               |
+----------------------------------------------------+---------------------------------------------+
