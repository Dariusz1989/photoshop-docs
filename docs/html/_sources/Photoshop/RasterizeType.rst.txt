.. _RasterizeType:

================================================
RasterizeType
================================================


Description
-----------



The type of the layer content to rasterize.




Static Properties
^^^^^^^^^^^^^^^^^

+--------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`ENTIRELAYER<RasterizeType.ENTIRELAYER>` readonly             | Rasterizes all vector data on the layer.                                                   |
+--------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`FILLCONTENT<RasterizeType.FILLCONTENT>` readonly             | Rasterizes the fill of a shape layer, leaving the vector mask.                             |
+--------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`LAYERCLIPPINGPATH<RasterizeType.LAYERCLIPPINGPATH>` readonly | Rasterizes the vector mask of a shape layer, turning it into a layer mask.                 |
+--------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`LINKEDLAYERS<RasterizeType.LINKEDLAYERS>` readonly           | Rasterizes the selected layers.                                                            |
+--------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`SHAPE<RasterizeType.SHAPE>` readonly                         | Rasterizes a shape layer.                                                                  |
+--------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`TEXTCONTENTS<RasterizeType.TEXTCONTENTS>` readonly           | Rasterizes the type on a type layer. Doesn't rasterize any other vector data on the layer. |
+--------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
