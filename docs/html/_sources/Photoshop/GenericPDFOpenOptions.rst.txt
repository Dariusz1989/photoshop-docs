.. _GenericPDFOpenOptions:

================================================
GenericPDFOpenOptions
================================================


Description
-----------

Options that can be specified when opening a generic PDF document.






Static Properties
^^^^^^^^^^^^^^^^^

+-------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------+
| :ref:`antiAlias<GenericPDFOpenOptions.antiAlias>`                       | If true, anti-aliasing is used.                                                                                           |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------+
| :ref:`bitsPerChannel<GenericPDFOpenOptions.bitsPerChannel>`             | The number of bits per channel.                                                                                           |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------+
| :ref:`constrainProportions<GenericPDFOpenOptions.constrainProportions>` | DEPRECATED.                                                                                                               |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------+
| :ref:`cropPage<GenericPDFOpenOptions.cropPage>`                         | The cropping method to use.                                                                                               |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------+
| :ref:`height<GenericPDFOpenOptions.height>`                             | DEPRECATED.                                                                                                               |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------+
| :ref:`mode<GenericPDFOpenOptions.mode>`                                 | The document mode.                                                                                                        |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------+
| :ref:`name<GenericPDFOpenOptions.name>`                                 | The name of the document.                                                                                                 |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------+
| :ref:`object<GenericPDFOpenOptions.object>`                             | The number of the 3d object to open.                                                                                      |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------+
| :ref:`page<GenericPDFOpenOptions.page>`                                 | The number of the page or image to open.                                                                                  |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------+
| :ref:`resolution<GenericPDFOpenOptions.resolution>`                     | The resolution of the document (in pixels per inch)                                                                       |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------+
| :ref:`suppressWarnings<GenericPDFOpenOptions.suppressWarnings>`         | If true, suppresses any warnings that may occur during opening.                                                           |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------+
| :ref:`use3DObjectNumber<GenericPDFOpenOptions.use3DObjectNumber>`       | If true, the value specified in the page property refers to a page number. If false, the value specifies an image number. |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------+
| :ref:`usePageNumber<GenericPDFOpenOptions.usePageNumber>`               | If true, the value specified in the page property refers to a page number. If false, the value specifies an image number. |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------+
| :ref:`width<GenericPDFOpenOptions.width>`                               | DEPRECATED.                                                                                                               |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------+
