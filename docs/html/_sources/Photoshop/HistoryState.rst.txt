================================================
HistoryState
================================================


Description
-----------

A version of the document stored automatically (and added to the history states collection, which preserves the document state each time the document is changed).

+-------------------------------------------------+-------------------------------------------+

+-------------------------------------------------+-------------------------------------------+
| :ref:`snapshot<HistoryState.snapshot>` readonly | If true, the history state is a snapshot. |
+-------------------------------------------------+-------------------------------------------+
| :ref:`typename<HistoryState.typename>` readonly | The class name of the object.             |
+-------------------------------------------------+-------------------------------------------+
