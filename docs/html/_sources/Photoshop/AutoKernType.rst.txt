================================================
AutoKernType
================================================


Description
-----------



The type of kerning to use for characters.




Static Properties
^^^^^^^^^^^^^^^^^

+-----------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`MANUAL<AutoKernType.MANUAL>` readonly   | Allows manual kerning.                                                                     |
+-----------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`METRICS<AutoKernType.METRICS>` readonly | Uses kern pairs, which contain information about the spacing of specific pairs of letters. |
+-----------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`OPTICAL<AutoKernType.OPTICAL>` readonly | Adjusts the spacing between adjacent characters based on their shapes.                     |
+-----------------------------------------------+--------------------------------------------------------------------------------------------+
