.. _Preferences:

================================================
Preferences
================================================


Description
-----------

Options to define for the preferences property of the application object, basically equivalent to selecting Edit > Preferences (Windows) or Photoshop > Preferences in the Adobe Photoshop application.




Properties
^^^^^^^^^^

+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`additionalPluginFolder<Preferences.additionalPluginFolder>`         | The path to the additional plug-in folder. Valid only when 'use additional plugin folder' = true.                              |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`appendExtension<Preferences.appendExtension>`                       | Save files with extensions on Windows.                                                                                         |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`askBeforeSavingLayeredTIFF<Preferences.askBeforeSavingLayeredTIFF>` | If true, asks the user to verify layer preservation options when saving a file in TIFF format.                                 |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`autoUpdateOpenDocuments<Preferences.autoUpdateOpenDocuments>`       | If true, automatically updates open documents.                                                                                 |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`beepWhenDone<Preferences.beepWhenDone>`                             | If true, alerts the user when a process finishes.                                                                              |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`colorChannelsInColor<Preferences.colorChannelsInColor>`             | If true, displays component channels in the Channels palette in color.                                                         |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`colorPicker<Preferences.colorPicker>`                               | The color picker to use.                                                                                                       |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`columnGutter<Preferences.columnGutter>`                             | The gutter of columns (in points)                                                                                              |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`columnWidth<Preferences.columnWidth>`                               | The width of columns (in points)                                                                                               |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`createFirstSnapshot<Preferences.createFirstSnapshot>`               | If true, automatically makes the first snapshot when a new document is created.                                                |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`dynamicColorSliders<Preferences.dynamicColorSliders>`               | If true, dynamic color sliders appear in the Color palette.                                                                    |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`editLogItems<Preferences.editLogItems>`                             | Options for edit log items.                                                                                                    |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`exportClipboard<Preferences.exportClipboard>`                       | If true, retains Adobe Photoshop contents on the clipboard after you exit the application.                                     |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`fontPreviewSize<Preferences.fontPreviewSize>`                       | Show font previews in the type tool font menus.                                                                                |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`fullSizePreview<Preferences.fullSizePreview>`                       | If true, shows the image preview as a full size image.                                                                         |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`gamutWarningOpacity<Preferences.gamutWarningOpacity>`               | The opacity (as a percentage) of the warning color for out-of-gamut colors. Range: 0 to 100.                                   |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`gridSize<Preferences.gridSize>`                                     | The size of grid squares.                                                                                                      |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`gridStyle<Preferences.gridStyle>`                                   | The formatting style for non-printing grid lines.                                                                              |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`gridSubDivisions<Preferences.gridSubDivisions>`                     | The value by which to subdivide the grid.                                                                                      |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`guideStyle<Preferences.guideStyle>`                                 | The formatting style for non-printing guide lines.                                                                             |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`iconPreview<Preferences.iconPreview>`                               | If true, shows the image preview as a thumbnail.                                                                               |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`imageCacheForHistograms<Preferences.imageCacheForHistograms>`       | If true, shows the current image cache used to create the histogram.                                                           |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`imageCacheLevels<Preferences.imageCacheLevels>`                     | The number of images to hold in the cache. Range: 1 to 8.                                                                      |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`imagePreviews<Preferences.imagePreviews>`                           | The behavior mode to use when saving files.                                                                                    |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`interpolation<Preferences.interpolation>`                           | The method to use to assign color values to any new pixels created when an image is resampled or resized.                      |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`keyboardZoomResizesWindows<Preferences.keyboardZoomResizesWindows>` | If true, automatically resizes the window when zooming in or out using keyboard shortcuts.                                     |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`macOSThumbnail<Preferences.macOSThumbnail>`                         | If true, creates a thumbnail when saving the image in Mac OS.                                                                  |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`maxRAMuse<Preferences.maxRAMuse>`                                   | Maximum percentage of available RAM used by Photoshop.                                                                         |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`maximizeCompatibility<Preferences.maximizeCompatibility>`           | The behavior to use to check whether to maximize compatibility when opening Adobe Photoshop (PSD) files.                       |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`nonLinearHistory<Preferences.nonLinearHistory>`                     | If true, allows non-linear history.                                                                                            |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`numberOfHistoryStates<Preferences.numberOfHistoryStates>`           | The number of history states to preserve. Range: 1 to 100.                                                                     |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`otherCursors<Preferences.otherCursors>`                             | The type of pointer to use.                                                                                                    |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`paintingCursors<Preferences.paintingCursors>`                       | The type of pointer to use.                                                                                                    |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`parent<Preferences.parent>` readonly                                | The object's container.                                                                                                        |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`pixelDoubling<Preferences.pixelDoubling>`                           | If true, halves the resolution or (doubles the size of pixels) to make previews display more quickly.                          |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`pointSize<Preferences.pointSize>`                                   | The point/pica size.                                                                                                           |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`recentFileListLength<Preferences.recentFileListLength>`             | The number of items in the recent file list. Range: 0 to 30.                                                                   |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`rulerUnits<Preferences.rulerUnits>`                                 | The unit that the scripting system uses when receiving and returning values.                                                   |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`saveLogItems<Preferences.saveLogItems>`                             | Options for saving the history items.                                                                                          |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`saveLogItemsFile<Preferences.saveLogItemsFile>`                     | File to save the history log.                                                                                                  |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`savePaletteLocations<Preferences.savePaletteLocations>`             | If true, makes new palette locations the default location.                                                                     |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`showAsianTextOptions<Preferences.showAsianTextOptions>`             | If true, Asian text options are displayed in the Paragraph palette.                                                            |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`showEnglishFontNames<Preferences.showEnglishFontNames>`             | If true, Asian font names are listed in English.                                                                               |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`showSliceNumber<Preferences.showSliceNumber>`                       | If true, displays slice numbers in the document window when using the Slice tool.                                              |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`showToolTips<Preferences.showToolTips>`                             | If true, pop-up definitions are displayed on mouseover.                                                                        |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`smartQuotes<Preferences.smartQuotes>`                               | If true, curly quote marks are used.                                                                                           |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`textFontSize<Preferences.textFontSize>`                             | Size of the small font used in panels and dialogs.                                                                             |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`typeUnits<Preferences.typeUnits>`                                   | The unit type-size that the numeric inputs are assumed to represent.                                                           |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`typename<Preferences.typename>` readonly                            | The class name of the object.                                                                                                  |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`useAdditionalPluginFolder<Preferences.useAdditionalPluginFolder>`   | If true, uses an additional folder for compatible plug-ins stored with a different application.                                |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`useDiffusionDither<Preferences.useDiffusionDither>`                 | If true, uses diffusion dither.                                                                                                |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`useHistoryLog<Preferences.useHistoryLog>`                           | Turn on and off the history logging.                                                                                           |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`useLowerCaseExtension<Preferences.useLowerCaseExtension>`           | If true, the file extension is lowercase.                                                                                      |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`useShiftKeyForToolSwitch<Preferences.useShiftKeyForToolSwitch>`     | If true, enables cycling through a set of hidden tools.                                                                        |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`useVideoAlpha<Preferences.useVideoAlpha>`                           | If true, enables Adobe Photoshop to send transparency information to your computer's video board. (Requires hardware support.) |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
| :ref:`windowsThumbnail<Preferences.windowsThumbnail>`                     | If true, creates a thumbnail when saving the image in Windows. (Requires hardware support.)                                    |
+---------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------------------------+
