.. _EPSSaveOptions:

================================================
EPSSaveOptions
================================================


Description
-----------

Options for saving a document in EPS format.






Static Properties
^^^^^^^^^^^^^^^^^

+---------------------------------------------------------------------+-----------------------------------------------------------------------------------------------------------------+
| :ref:`embedColorProfile<EPSSaveOptions.embedColorProfile>` readonly | If true, the color profile is embedded in the document.                                                         |
+---------------------------------------------------------------------+-----------------------------------------------------------------------------------------------------------------+
| :ref:`encoding<EPSSaveOptions.encoding>` readonly                   | The type of encoding to use for the document.                                                                   |
+---------------------------------------------------------------------+-----------------------------------------------------------------------------------------------------------------+
| :ref:`halftoneScreen<EPSSaveOptions.halftoneScreen>` readonly       | If true, includes the halftone screen.                                                                          |
+---------------------------------------------------------------------+-----------------------------------------------------------------------------------------------------------------+
| :ref:`interpolation<EPSSaveOptions.interpolation>` readonly         | If true, uses image interpolation.                                                                              |
+---------------------------------------------------------------------+-----------------------------------------------------------------------------------------------------------------+
| :ref:`preview<EPSSaveOptions.preview>` readonly                     | The type of preview.                                                                                            |
+---------------------------------------------------------------------+-----------------------------------------------------------------------------------------------------------------+
| :ref:`psColorManagement<EPSSaveOptions.psColorManagement>` readonly | If true, uses PostScript color management.                                                                      |
+---------------------------------------------------------------------+-----------------------------------------------------------------------------------------------------------------+
| :ref:`transferFunction<EPSSaveOptions.transferFunction>` readonly   | If true, includes the transfer functions in the document to compensate for dot gain between the image and film. |
+---------------------------------------------------------------------+-----------------------------------------------------------------------------------------------------------------+
| :ref:`transparentWhites<EPSSaveOptions.transparentWhites>` readonly | If true, displays white areas as transparent. Valid only for documents in BitMap mode.                          |
+---------------------------------------------------------------------+-----------------------------------------------------------------------------------------------------------------+
| :ref:`vectorData<EPSSaveOptions.vectorData>` readonly               | If true, includes vector data. Valid only when the document contains vector data (un-rasterized text).          |
+---------------------------------------------------------------------+-----------------------------------------------------------------------------------------------------------------+
