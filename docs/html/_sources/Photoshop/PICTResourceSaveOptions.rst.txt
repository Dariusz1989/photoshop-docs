.. _PICTResourceSaveOptions:

================================================
PICTResourceSaveOptions
================================================


Description
-----------

Options for saving a document as a PICT resource file.






Static Properties
^^^^^^^^^^^^^^^^^

+---------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`alphaChannels<PICTResourceSaveOptions.alphaChannels>`         | If true, the alpha channels are saved.                  |
+---------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`compression<PICTResourceSaveOptions.compression>`             | The compression type.                                   |
+---------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`embedColorProfile<PICTResourceSaveOptions.embedColorProfile>` | If true, the color profile is embedded in the document. |
+---------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`name<PICTResourceSaveOptions.name>`                           | The name of PICT resource.                              |
+---------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`resolution<PICTResourceSaveOptions.resolution>`               | The number of bits per pixel.                           |
+---------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`resourceID<PICTResourceSaveOptions.resourceID>`               | The ID of the PICT resource.                            |
+---------------------------------------------------------------------+---------------------------------------------------------+
