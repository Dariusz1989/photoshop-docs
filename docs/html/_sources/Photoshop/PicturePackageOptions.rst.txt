================================================
PicturePackageOptions
================================================


Description
-----------

Options that can be specified for a Picture Package.






Static Properties
^^^^^^^^^^^^^^^^^

+---------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`content<PicturePackageOptions.content>`           | The content information.                                              |
+---------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`flatten<PicturePackageOptions.flatten>`           | If true, flattens all layers in the final document.                   |
+---------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`font<PicturePackageOptions.font>`                 | The font used for security text.                                      |
+---------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`fontSize<PicturePackageOptions.fontSize>`         | The font size.                                                        |
+---------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`layout<PicturePackageOptions.layout>`             | The layout to use to generate the picture package.                    |
+---------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`mode<PicturePackageOptions.mode>`                 | The document color mode.                                              |
+---------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`opacity<PicturePackageOptions.opacity>`           | The web page security text opacity (as a percentage) Range: 0 to 100. |
+---------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`resolution<PicturePackageOptions.resolution>`     | The resolution of the document (in pixels per inch)                   |
+---------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`text<PicturePackageOptions.text>`                 | The picture package custom text.                                      |
+---------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`textColor<PicturePackageOptions.textColor>`       | The color of the security text.                                       |
+---------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`textPosition<PicturePackageOptions.textPosition>` | The position of the security text.                                    |
+---------------------------------------------------------+-----------------------------------------------------------------------+
| :ref:`textRotate<PicturePackageOptions.textRotate>`     | The orientation of the security text.                                 |
+---------------------------------------------------------+-----------------------------------------------------------------------+
