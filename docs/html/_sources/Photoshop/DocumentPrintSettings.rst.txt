DocumentPrintSettings
================================================


Description
-----------

Print settings for a document.

+-------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------+
| :ref:`colorHandling<DocumentPrintSettings.colorHandling>`         | Color handling.                                                                                                   |

| :ref:`copies<DocumentPrintSettings.copies>`                       | Number of copies.                                                                                                 |
+-------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------+
| :ref:`cornerCropMarks<DocumentPrintSettings.cornerCropMarks>`     | Print corner crop marks.                                                                                          |

+-------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------+
| :ref:`flip<DocumentPrintSettings.flip>`                           | Flip the image horizontally.                                                                                      |
+-------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------+
| :ref:`hardProof<DocumentPrintSettings.hardProof>`                 | Print a hard proof.                                                                                               |

| :ref:`interpolate<DocumentPrintSettings.interpolate>`             |                                                                                                                   |

| :ref:`labels<DocumentPrintSettings.labels>`                       | Prints the document title.                                                                                        |

| :ref:`mapBlack<DocumentPrintSettings.mapBlack>`                   | Map blacks.                                                                                                       |
+-------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------+
| :ref:`negative<DocumentPrintSettings.negative>`                   | Invert the image colors.                                                                                          |
+-------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------+
| :ref:`parent<DocumentPrintSettings.parent>` readonly              | The object's container.                                                                                           |
+-------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------+
| :ref:`posX<DocumentPrintSettings.posX>` readonly                  | X position of image on page.                                                                                      |
+-------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------+
| :ref:`posY<DocumentPrintSettings.posY>` readonly                  | Y position of image on page.                                                                                      |
+-------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------+
| :ref:`printBorder<DocumentPrintSettings.printBorder>`             | Width of the print border.                                                                                        |
+-------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------+
| :ref:`printSpace<DocumentPrintSettings.printSpace>`               | Color space for printer. Can be nothing (meaning same as source) or a string specifying a specific color profile. |

| :ref:`printerName<DocumentPrintSettings.printerName>`             | Name of printer.                                                                                                  |

| :ref:`registrationMarks<DocumentPrintSettings.registrationMarks>` | Print registration marks.                                                                                         |

+-------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------+
