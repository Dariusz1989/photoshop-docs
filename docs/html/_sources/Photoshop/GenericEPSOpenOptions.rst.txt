.. _GenericEPSOpenOptions:

================================================
GenericEPSOpenOptions
================================================


Description
-----------

Options that can be specified when opening a generic EPS document.






Static Properties
^^^^^^^^^^^^^^^^^

+----------------------------------------------------------------------------------+-----------------------------------------------------+
| :ref:`antiAlias<GenericEPSOpenOptions.antiAlias>` readonly                       | If true, anti-aliasing is used.                     |
+----------------------------------------------------------------------------------+-----------------------------------------------------+
| :ref:`constrainProportions<GenericEPSOpenOptions.constrainProportions>` readonly | If true, the image proportions are constrained.     |
+----------------------------------------------------------------------------------+-----------------------------------------------------+
| :ref:`height<GenericEPSOpenOptions.height>` readonly                             | The image height.                                   |
+----------------------------------------------------------------------------------+-----------------------------------------------------+
| :ref:`mode<GenericEPSOpenOptions.mode>` readonly                                 | The document mode.                                  |
+----------------------------------------------------------------------------------+-----------------------------------------------------+
| :ref:`resolution<GenericEPSOpenOptions.resolution>` readonly                     | The resolution of the document (in pixels per inch) |
+----------------------------------------------------------------------------------+-----------------------------------------------------+
| :ref:`width<GenericEPSOpenOptions.width>` readonly                               | The image width.                                    |
+----------------------------------------------------------------------------------+-----------------------------------------------------+
