.. _GenericEPSOpenOptions:

================================================
GenericEPSOpenOptions
================================================


Description
-----------

Options that can be specified when opening a generic EPS document.






Static Properties
^^^^^^^^^^^^^^^^^

+-------------------------------------------------------------------------+-----------------------------------------------------+
| :ref:`antiAlias<GenericEPSOpenOptions.antiAlias>`                       | If true, anti-aliasing is used.                     |
+-------------------------------------------------------------------------+-----------------------------------------------------+
| :ref:`constrainProportions<GenericEPSOpenOptions.constrainProportions>` | If true, the image proportions are constrained.     |
+-------------------------------------------------------------------------+-----------------------------------------------------+
| :ref:`height<GenericEPSOpenOptions.height>`                             | The image height.                                   |
+-------------------------------------------------------------------------+-----------------------------------------------------+
| :ref:`mode<GenericEPSOpenOptions.mode>`                                 | The document mode.                                  |
+-------------------------------------------------------------------------+-----------------------------------------------------+
| :ref:`resolution<GenericEPSOpenOptions.resolution>`                     | The resolution of the document (in pixels per inch) |
+-------------------------------------------------------------------------+-----------------------------------------------------+
| :ref:`width<GenericEPSOpenOptions.width>`                               | The image width.                                    |
+-------------------------------------------------------------------------+-----------------------------------------------------+
