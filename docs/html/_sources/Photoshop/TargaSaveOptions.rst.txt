================================================
TargaSaveOptions
================================================


Description
-----------

Options for saving a document in TGA (Targa) format.






Static Properties
^^^^^^^^^^^^^^^^^

+--------------------------------------------------------+----------------------------------------+
| :ref:`alphaChannels<TargaSaveOptions.alphaChannels>`   | If true, the alpha channels are saved. |
+--------------------------------------------------------+----------------------------------------+
| :ref:`resolution<TargaSaveOptions.resolution>`         | The number of bits per pixel.          |
+--------------------------------------------------------+----------------------------------------+
| :ref:`rleCompression<TargaSaveOptions.rleCompression>` | If true, RLE compression is used.      |
+--------------------------------------------------------+----------------------------------------+
