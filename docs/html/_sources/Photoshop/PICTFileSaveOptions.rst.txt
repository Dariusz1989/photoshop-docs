================================================
PICTFileSaveOptions
================================================


Description
-----------

Options for saving a document in PICT format.






Static Properties
^^^^^^^^^^^^^^^^^

+--------------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`alphaChannels<PICTFileSaveOptions.alphaChannels>` readonly         | If true, the alpha channels are saved.                  |
+--------------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`compression<PICTFileSaveOptions.compression>` readonly             | The compression method.                                 |
+--------------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`embedColorProfile<PICTFileSaveOptions.embedColorProfile>` readonly | If true, the color profile is embedded in the document. |
+--------------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`resolution<PICTFileSaveOptions.resolution>` readonly               | The number of bits per pixel.                           |
+--------------------------------------------------------------------------+---------------------------------------------------------+
