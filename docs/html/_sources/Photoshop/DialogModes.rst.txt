.. _DialogModes:

================================================
DialogModes
================================================


Description
-----------



Controls whether Photoshop displays dialogs during scripts.




Static Properties
^^^^^^^^^^^^^^^^^

+------------------------------------------+--------------------------------------+
| :ref:`ALL<DialogModes.ALL>` readonly     | Show all dialogs.                    |
+------------------------------------------+--------------------------------------+
| :ref:`ERROR<DialogModes.ERROR>` readonly | Show only dialogs related to errors. |
+------------------------------------------+--------------------------------------+
| :ref:`NO<DialogModes.NO>` readonly       | Show no dialogs.                     |
+------------------------------------------+--------------------------------------+
