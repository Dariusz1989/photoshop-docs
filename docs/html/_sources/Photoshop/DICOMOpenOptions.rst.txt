.. _DICOMOpenOptions:

================================================
DICOMOpenOptions
================================================


Description
-----------

Options for opening a DICOM document.






Static Properties
^^^^^^^^^^^^^^^^^

+----------------------------------------------------+---------------------------------------------------+
| :ref:`anonymize<DICOMOpenOptions.anonymize>`       | If true, patient information is anonymized.       |
+----------------------------------------------------+---------------------------------------------------+
| :ref:`columns<DICOMOpenOptions.columns>`           | The number of columns in an n-up configuration.   |
+----------------------------------------------------+---------------------------------------------------+
| :ref:`reverse<DICOMOpenOptions.reverse>`           | If true, the image is inverted.                   |
+----------------------------------------------------+---------------------------------------------------+
| :ref:`rows<DICOMOpenOptions.rows>`                 | The number of rows in an n-up configuration.      |
+----------------------------------------------------+---------------------------------------------------+
| :ref:`showOverlays<DICOMOpenOptions.showOverlays>` | If true, overlays are shown (if present).         |
+----------------------------------------------------+---------------------------------------------------+
| :ref:`windowLevel<DICOMOpenOptions.windowLevel>`   | The contrast of the image in Houndsfield units.   |
+----------------------------------------------------+---------------------------------------------------+
| :ref:`windowWidth<DICOMOpenOptions.windowWidth>`   | The brightness of the image in Houndsfield units. |
+----------------------------------------------------+---------------------------------------------------+
