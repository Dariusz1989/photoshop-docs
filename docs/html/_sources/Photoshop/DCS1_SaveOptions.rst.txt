.. _DCS1_SaveOptions:

================================================
DCS1_SaveOptions
================================================


Description
-----------

Options for saving a document in Photoshop DCS 1.0 format.






Static Properties
^^^^^^^^^^^^^^^^^

+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`DCS<DCS1_SaveOptions.DCS>` readonly                             | The DCS type.                                                                                               |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`embedColorProfile<DCS1_SaveOptions.embedColorProfile>` readonly | If true, the color profile is embedded in the document.                                                     |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`encoding<DCS1_SaveOptions.encoding>` readonly                   | The type of encoding to use for the document.                                                               |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`halftoneScreen<DCS1_SaveOptions.halftoneScreen>` readonly       | If true, includes halftone screen.                                                                          |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`interpolation<DCS1_SaveOptions.interpolation>` readonly         | If true, image interpolation is used.                                                                       |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`preview<DCS1_SaveOptions.preview>` readonly                     | The type of preview.                                                                                        |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`transferFunction<DCS1_SaveOptions.transferFunction>` readonly   | If true, includes transfer functions in the document to compensate for dot gain between the image and film. |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
| :ref:`vectorData<DCS1_SaveOptions.vectorData>` readonly               | If true, includes vector data. Valid only if the document contains vector data (un-rasterized text).        |
+-----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------+
