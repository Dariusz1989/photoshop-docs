   :ref:`LayerComp` **add** (:ref:`string` **name**, :ref:`string` **comment**, bool **appearance**, bool **position**, bool **visibility**)


Parameters
----------

+----------------+------------------------------------------------------------------+
| **name**       | The name of the layer comp.                                      |
+----------------+------------------------------------------------------------------+
| **comment**    | The description of the layer comp.                               |
+----------------+------------------------------------------------------------------+
| **appearance** | If true, uses the layer appearance or style for this layer comp. |
