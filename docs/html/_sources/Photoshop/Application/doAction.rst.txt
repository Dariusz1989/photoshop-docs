.. _Application.doAction:

================================================
Application.doAction
================================================

   void **doAction** (:ref:`string` **action**, :ref:`string` **from**)


Parameters
----------

+------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **action** | The name of the action to play. (Note that the action name is case-sensitive and must match the name as it appears in the Actions palette.)                                    |
+------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **from**   | The name of the action set containing the action being played. (Note that the Action Set name is case-sensitive and must match the name as it appears in the Actions palette.) |
+------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+



Description
-----------

Plays the specified action from the Actions palette.
