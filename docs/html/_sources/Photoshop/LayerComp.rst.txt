.. _LayerComp:

================================================
LayerComp
================================================


Description
-----------

A snapshot of a state of the layers in a document (can be used to view different page layouts or compostions).




Properties
^^^^^^^^^^

+----------------------------------------------+---------------------------------------------------------+
| :ref:`appearance<LayerComp.appearance>`      | If true, uses layer appearance (layer styles) settings. |
+----------------------------------------------+---------------------------------------------------------+
| :ref:`comment<LayerComp.comment>`            | The description of the layer comp.                      |
+----------------------------------------------+---------------------------------------------------------+
| :ref:`name<LayerComp.name>`                  | The name of the layer comp.                             |
+----------------------------------------------+---------------------------------------------------------+
| :ref:`parent<LayerComp.parent>` readonly     | The object's container.                                 |
+----------------------------------------------+---------------------------------------------------------+
| :ref:`position<LayerComp.position>`          | If true, uses layer position.                           |
+----------------------------------------------+---------------------------------------------------------+
| :ref:`selected<LayerComp.selected>` readonly | If true, the layer comp is currently selected.          |
+----------------------------------------------+---------------------------------------------------------+
| :ref:`typename<LayerComp.typename>` readonly | The class name of the object.                           |
+----------------------------------------------+---------------------------------------------------------+
| :ref:`visibility<LayerComp.visibility>`      | If true, the layer comp is visible.                     |
+----------------------------------------------+---------------------------------------------------------+







Methods
^^^^^^^

+-----------------------------------------------+------------------------------------------------------------+
| :ref:`add<LayerComp.add>`                     | Adds an element.                                           |
+-----------------------------------------------+------------------------------------------------------------+
| :ref:`apply<LayerComp.apply>`                 | Applies the layer comp to the document.                    |
+-----------------------------------------------+------------------------------------------------------------+
| :ref:`recapture<LayerComp.recapture>`         | Recaptures the current layer state(s) for this layer comp. |
+-----------------------------------------------+------------------------------------------------------------+
| :ref:`remove<LayerComp.remove>`               | Deletes this object.                                       |
+-----------------------------------------------+------------------------------------------------------------+
| :ref:`removeAll<LayerComp.removeAll>`         | Deletes all elements.                                      |
+-----------------------------------------------+------------------------------------------------------------+
| :ref:`resetFromComp<LayerComp.resetFromComp>` | Resets the layer comp state to the document state.         |
+-----------------------------------------------+------------------------------------------------------------+
