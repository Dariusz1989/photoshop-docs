.. _JavaScriptExecutionMode:

================================================
JavaScriptExecutionMode
================================================


Description
-----------



When should a JavaScript debugger be shown.




Static Properties
^^^^^^^^^^^^^^^^^

+------------------------------------------------------------------------+-----------------------------------------------------------------------------------------------+
| :ref:`BEFORERUNNING<JavaScriptExecutionMode.BEFORERUNNING>` readonly   | Show the JavaScript debugger when the first line of the JavaScript executes.                  |
+------------------------------------------------------------------------+-----------------------------------------------------------------------------------------------+
| :ref:`NEVER<JavaScriptExecutionMode.NEVER>` readonly                   | Never show the JavaScript debugger. Treat runtime errors by throwing a JavaScript exceptions. |
+------------------------------------------------------------------------+-----------------------------------------------------------------------------------------------+
| :ref:`ONRUNTIMEERROR<JavaScriptExecutionMode.ONRUNTIMEERROR>` readonly | Show the JavaScript debugger if a runtime error occurs.                                       |
+------------------------------------------------------------------------+-----------------------------------------------------------------------------------------------+
