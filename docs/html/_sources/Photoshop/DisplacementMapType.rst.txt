================================================
DisplacementMapType
================================================


Description
-----------



Describes how the displacement map fits the image if the image is not the same size as the map.




Static Properties
^^^^^^^^^^^^^^^^^

+----------------------------------------------------------------+------------------------------------------------------------+
| :ref:`STRETCHTOFIT<DisplacementMapType.STRETCHTOFIT>` readonly | The map is resized.                                        |
+----------------------------------------------------------------+------------------------------------------------------------+
| :ref:`TILE<DisplacementMapType.TILE>` readonly                 | The selection is filled by repeating the map in a pattern. |
+----------------------------------------------------------------+------------------------------------------------------------+
