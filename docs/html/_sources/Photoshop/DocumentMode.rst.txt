DocumentMode
================================================


Description
-----------



The document's color mode.




Static Properties
^^^^^^^^^^^^^^^^^

+---------------------------------------------------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :ref:`BITMAP<DocumentMode.BITMAP>` readonly             | Bitmap, which uses one of two color values (black or white) to represent the pixels in an image.                                                                                 |
+---------------------------------------------------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :ref:`CMYK<DocumentMode.CMYK>` readonly                 | CMYK.                                                                                                                                                                            |
+---------------------------------------------------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :ref:`DUOTONE<DocumentMode.DUOTONE>` readonly           | Duotone mode, which creates monotone, duotone (two-color), tritone (three-color), and quadtone (four-color) grayscale images using one to four custom inks.                      |
+---------------------------------------------------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :ref:`GRAYSCALE<DocumentMode.GRAYSCALE>` readonly       | Grayscale.                                                                                                                                                                       |
+---------------------------------------------------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :ref:`INDEXEDCOLOR<DocumentMode.INDEXEDCOLOR>` readonly | Indexed color, in which the number of colors in the image is at most 256, the standard number of colors supported by the GIF and PNG-8 formats and many multimedia applications. |
+---------------------------------------------------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :ref:`LAB<DocumentMode.LAB>` readonly                   | Lab.                                                                                                                                                                             |
+---------------------------------------------------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :ref:`MULTICHANNEL<DocumentMode.MULTICHANNEL>` readonly | Image with multiple color channels.                                                                                                                                              |
+---------------------------------------------------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :ref:`RGB<DocumentMode.RGB>` readonly                   | RGB.                                                                                                                                                                             |
+---------------------------------------------------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
