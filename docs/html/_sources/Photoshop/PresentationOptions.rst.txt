.. _PresentationOptions:

================================================
PresentationOptions
================================================


Description
-----------

Options for the PDF presentation command.






Static Properties
^^^^^^^^^^^^^^^^^

+----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------+
| :ref:`PDFFileOptions<PresentationOptions.PDFFileOptions>` readonly   | Options for creating the PDF file.                                                                                      |
+----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------+
| :ref:`autoAdvance<PresentationOptions.autoAdvance>` readonly         | If true, the presentation auto advances.                                                                                |
+----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------+
| :ref:`includeFilename<PresentationOptions.includeFilename>` readonly | If true, includes the file name for the image.                                                                          |
+----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------+
| :ref:`interval<PresentationOptions.interval>` readonly               | The amount of time (in seconds) before auto advancing the view. Valid only when 'auto advance' is true. Range: 1 to 60. |
+----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------+
| :ref:`loop<PresentationOptions.loop>` readonly                       | If true, the presentation loops after the last page.                                                                    |
+----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------+
| :ref:`magnification<PresentationOptions.magnification>` readonly     | The magnification type when viewing the image.                                                                          |
+----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------+
| :ref:`presentation<PresentationOptions.presentation>` readonly       | If true, the file type is presentation. If false, the file type is Multi-Page document.                                 |
+----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------+
| :ref:`transition<PresentationOptions.transition>` readonly           | The image transition type.                                                                                              |
+----------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------+
