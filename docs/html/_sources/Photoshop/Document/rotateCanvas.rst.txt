Document.rotateCanvas
================================================

   void **rotateCanvas** (:ref:`number` **angle**)


Parameters
----------

+-----------+----------------------------------------------------------------------------------------------------------------------------------------+
| **angle** | The number of degrees to rotate. A positive angle rotates the canvas clockwise; a negative value rotates the canvas counter-clockwise. |
+-----------+----------------------------------------------------------------------------------------------------------------------------------------+



Description
-----------
