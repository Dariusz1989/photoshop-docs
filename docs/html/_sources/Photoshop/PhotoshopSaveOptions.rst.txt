.. _PhotoshopSaveOptions:

================================================
PhotoshopSaveOptions
================================================


Description
-----------

Options for saving a Photoshop document.






Static Properties
^^^^^^^^^^^^^^^^^

+------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`alphaChannels<PhotoshopSaveOptions.alphaChannels>`         | If true, the alpha channels are saved.                  |
+------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`annotations<PhotoshopSaveOptions.annotations>`             | If true, the annotations are saved.                     |
+------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`embedColorProfile<PhotoshopSaveOptions.embedColorProfile>` | If true, the color profile is embedded in the document. |
+------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`layers<PhotoshopSaveOptions.layers>`                       | If true, the layers are saved.                          |
+------------------------------------------------------------------+---------------------------------------------------------+
| :ref:`spotColors<PhotoshopSaveOptions.spotColors>`               | If true, spot colors are saved.                         |
+------------------------------------------------------------------+---------------------------------------------------------+
