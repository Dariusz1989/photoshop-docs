Function
================================================


Description
-----------

Wraps a built-in or JavaScript function.




Properties
^^^^^^^^^^

+-----------------------------------------+-----------------------------------------------+
| :ref:`arguments<Function.arguments>`    | The function arguments, packed into an array. |
+-----------------------------------------+-----------------------------------------------+
| :ref:`arity<Function.arity>` readonly   | The number of formal arguments.               |
+-----------------------------------------+-----------------------------------------------+
| :ref:`length<Function.length>` readonly | The number of formal arguments.               |
+-----------------------------------------+-----------------------------------------------+
| :ref:`name<Function.name>` readonly     | The function name.                            |
+-----------------------------------------+-----------------------------------------------+

| :ref:`Function<Function.Function>` |  |

Methods
^^^^^^^

+------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------+
| :ref:`apply<Function.apply>`       | Apply a this object and an argument list to a function.                                                                                     |
+------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------+
| :ref:`call<Function.call>`         | Apply a this object and arguments to a function.                                                                                            |
+------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------+
| :ref:`toSource<Function.toSource>` | Creates a string representation of this object that can be fed back to eval() to re-create an object. Works only with JavaScript functions. |
+------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------+
| :ref:`toString<Function.toString>` | Returns the function definition as a string.                                                                                                |
+------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------+
