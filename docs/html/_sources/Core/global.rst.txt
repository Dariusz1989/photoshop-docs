.. _global:

================================================
global
================================================


Description
-----------

The global methods and properties for ExtendScript.




Properties
^^^^^^^^^^

+------------------------------------+----------------------------------------------------------------------------------------------------------------------------+
| :ref:`Infinity<global.Infinity>`   | The Infinity global property is a predefined variable with the value for infinity.                                         |
+------------------------------------+----------------------------------------------------------------------------------------------------------------------------+
| :ref:`NaN<global.NaN>`             | The NaN global property is a predefined variable with the value NaN (Not-a-Number), as specified by the IEEE-754 standard. |
+------------------------------------+----------------------------------------------------------------------------------------------------------------------------+
| :ref:`undefined<global.undefined>` | This global property is a predefined variable with the value for an undefined value.                                       |
+------------------------------------+----------------------------------------------------------------------------------------------------------------------------+







Methods
^^^^^^^

+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`alert<global.alert>`                                   | Displays an alert box                                                                      |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`confirm<global.confirm>`                               | Displays an alert box with Yes and No buttons; returns true for Yes                        |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`decodeURI<global.decodeURI>`                           | Decodes a string created with encodeURI().                                                 |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`decodeURIComponent<global.decodeURIComponent>`         | Decodes a string created with encodeURIComponent().                                        |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`encodeURI<global.encodeURI>`                           | Encodes a string after RFC2396.                                                            |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`encodeURIComponent<global.encodeURIComponent>`         | Encodes a string after RFC2396.                                                            |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`escape<global.escape>`                                 | Creates a URL-encoded string from aString.                                                 |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`eval<global.eval>`                                     | Evaluates its argument as a JavaScript script, and returns the result of evaluation.       |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`isFinite<global.isFinite>`                             | Evaluates an expression and reports whether the result is a finite number.                 |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`isNaN<global.isNaN>`                                   | Evaluates an expression and reports whether the result is "Not-a-Number" (NaN).            |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`isXMLName<global.isXMLName>`                           | Returns true if the supplied string is a valid XML name.                                   |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`localize<global.localize>`                             | Localizes a ZString-encoded string and merges additional arguments into the string.        |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`parseFloat<global.parseFloat>`                         | Extracts a floating-point number from a string.                                            |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`parseInt<global.parseInt>`                             | Extracts an integer from a string.                                                         |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`prompt<global.prompt>`                                 | Displays a dialog allowing the user to enter text                                          |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`setDefaultXMLNamespace<global.setDefaultXMLNamespace>` | Defines the default XML namespace.                                                         |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`unescape<global.unescape>`                             | Translates URL-encoded string into a regular string, and returns that string.              |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`uneval<global.uneval>`                                 | Creates a source code representation of the supplied argument, and returns it as a string. |
+--------------------------------------------------------------+--------------------------------------------------------------------------------------------+
