.. _global:

================================================
global
================================================


Description
-----------

The global methods and properties for ExtendScript.




Properties
^^^^^^^^^^

+---------------------------------------------+----------------------------------------------------------------------------------------------------------------------------+
| :ref:`Infinity<global.Infinity>` readonly   | The Infinity global property is a predefined variable with the value for infinity.                                         |
+---------------------------------------------+----------------------------------------------------------------------------------------------------------------------------+
| :ref:`NaN<global.NaN>` readonly             | The NaN global property is a predefined variable with the value NaN (Not-a-Number), as specified by the IEEE-754 standard. |
+---------------------------------------------+----------------------------------------------------------------------------------------------------------------------------+
| :ref:`undefined<global.undefined>` readonly | This global property is a predefined variable with the value for an undefined value.                                       |
+---------------------------------------------+----------------------------------------------------------------------------------------------------------------------------+







Methods
^^^^^^^

+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`alert<global.alert>` readonly                                   | Displays an alert box                                                                      |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`confirm<global.confirm>` readonly                               | Displays an alert box with Yes and No buttons; returns true for Yes                        |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`decodeURI<global.decodeURI>` readonly                           | Decodes a string created with?encodeURI().                                                 |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`decodeURIComponent<global.decodeURIComponent>` readonly         | Decodes a string created with?encodeURIComponent().                                        |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`encodeURI<global.encodeURI>` readonly                           | Encodes a string after RFC2396.                                                            |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`encodeURIComponent<global.encodeURIComponent>` readonly         | Encodes a string after RFC2396.                                                            |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`escape<global.escape>` readonly                                 | Creates a URL-encoded string from aString.                                                 |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`eval<global.eval>` readonly                                     | Evaluates its argument as a JavaScript script, and returns the result of evaluation.       |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`isFinite<global.isFinite>` readonly                             | Evaluates an expression and reports whether the result is a finite number.                 |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`isNaN<global.isNaN>` readonly                                   | Evaluates an expression and reports whether the result is "Not-a-Number" (NaN).            |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`isXMLName<global.isXMLName>` readonly                           | Returns true if the supplied string is a valid XML name.                                   |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`localize<global.localize>` readonly                             | Localizes a ZString-encoded string and merges additional arguments into the string.        |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`parseFloat<global.parseFloat>` readonly                         | Extracts a floating-point number from a string.                                            |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`parseInt<global.parseInt>` readonly                             | Extracts an integer from a string.                                                         |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`prompt<global.prompt>` readonly                                 | Displays a dialog allowing the user to enter text                                          |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`setDefaultXMLNamespace<global.setDefaultXMLNamespace>` readonly | Defines the default XML namespace.                                                         |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`unescape<global.unescape>` readonly                             | Translates URL-encoded string into a regular string, and returns that string.              |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
| :ref:`uneval<global.uneval>` readonly                                 | Creates a source code representation of the supplied argument, and returns it as a string. |
+-----------------------------------------------------------------------+--------------------------------------------------------------------------------------------+
