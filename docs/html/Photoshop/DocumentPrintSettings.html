DocumentPrintSettings¶

Description¶

Print settings for a document.

Properties¶

activePrinter Currently selected printer.
backgroundColor Background color of page.
bleedWidth Bleed width.
caption Description field from File Info.
centerCropMarks Print center crop marks.
colorBars Print color calibration bars.
colorHandling Color handling.
copies Number of copies.
cornerCropMarks Print corner crop marks.
docPosition readonly Position of image when printing.
flip Flip the image horizontally.
hardProof Print a hard proof.
interpolate  
labels Prints the document title.
mapBlack Map blacks.
negative Invert the image colors.
parent readonly The object’s container.
posX readonly X position of image on page.
posY readonly Y position of image on page.
printBorder Width of the print border.
printSpace Color space for printer. Can be nothing (meaning same as source) or a string specifying a specific color profile.
printerName Name of printer.
printers readonly List of available printers.
registrationMarks Print registration marks.
renderIntent Color conversion intent when print space is different from the source space.
scale readonly Scale of image on page.
typename readonly The class name of the object.
vectorData Include vector data.

Methods¶

setPagePosition Set the position of the image on the page.
