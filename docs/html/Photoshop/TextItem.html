TextItem¶

Description¶

The text object contained in an art layer.

Properties¶

alternateLigatures If true, alternate ligatures are used.
antiAliasMethod The method of anti-aliasing to use.
autoKerning Options for auto kerning.
autoLeadingAmount The percentage to use for auto leading. Range: 0.01 to 5000.00.
baselineShift The amount of baseline offset of text.
capitalization The case of the text.
color The text color.
contents The actual text in the layer.
desiredGlyphScaling The desired amount (as a percentage) to scale the horizontal size of the text letters. Range: 50 - 200; at 100, the width of characters is not scaled. Valid only for justified text.
desiredLetterScaling The amount of space (as a percentage) between letters. Range: 100 - 500; at 0, no space is added between letters. Valid only for justified text.
desiredWordScaling The amount (as a percentage) of space between words. Range: 0 -1000; at 100, no additional space is added between words. Valid only for justified text.
direction The text orientation.
fauxBold If true, faux bold is used.
fauxItalic If true, faux italic is used.
firstLineIndent The amount to indent the first line of paragraphs. Range: -1296 to 1296.
font The text face of the character.
hangingPuntuation If true, uses Roman hanging punctuation.
height The height of the bounding box for paragraph text.
horizontalScale Character scaling (horizontal) in proportion to horizontal scale. Range: 0 - 1000 as a percentage.
hyphenLimit The maximum number of consecutive lines that can end with a hyphenated word.
hyphenateAfterFirst The number of letters after which hyphenation in word wrap is allowed. Range: 1 to 15.
hyphenateBeforeLast The number of letters before which hyphenation in word wrap is allowed. Range: 1 to 15.
hyphenateCapitalWords If true, capitalized words can be hyphenated.
hyphenateWordsLongerThan The minimum number of letters a word must have in order for hyphenation in word wrap to be allowed. Range: 2 to 25.
hyphenation If true, hyphenation is used.
hyphenationZone The distance at the end of a line that will cause a word to break in unjustified type. Range: 0 - 720 picas.
justification The paragraph justification.
kind The type of text.
language The language.
leading The leading amount.
leftIndent The amount to indent text from the left. Range: -1296 to 1296.
ligatures If true, ligatures are used.
maximumGlyphScaling The maximum amount (as a percentage) to scale the horizontal size of the text letters. Range: 50 - 200; at 100, the width of characters is not scaled. Valid only for justified text.
maximumLetterScaling The maximum amount (as a percentage) of space between letters. Range: 100 - 500; at 0, no space is added between letters. Valid only for justified text.
maximumWordScaling The maximum amount (as a percentage) of space between words (0 -1000; at 100, no additional space is added between words). Valid only for justified text.
minimumGlyphScaling The minimum amount (as a percentage) to scale the horizontal size of the text letters. Range: 50 - 200; at 100, the width of characters is not scaled. Valid only for justified text.
minimumLetterScaling The minimum amount of space (as a percentage) between letters. Range: 100 to 500; at 0, no space is added between letters. Valid only for justified text.
minimumWordScaling The minimum amount (as a percentage) of space between words. Range: 0 -1000; at 100, no additional space is added between words. Valid only for justified text.
noBreak If true, words are not allowed to break at the end of a line. When enacted on large amounts of consecutive characters, can prevent word wrap and thus may prevent some text from appearing on the screen.
oldStyle If true, old style is used.
parent readonly The object’s container.
position The position of the origin for the text. The array must contain two values. Setting this property is basically equivalent to clicking the text tool at a point in the document to create the point of origin for text.
rightIndent The amount to indent text from the right. Range: -1296 to 1296.
size The font size in points.
spaceAfter The amount of space after each paragraph. Range: -1296 to 1296.
spaceBefore The amount of space before each paragraph. Range: -1296 to 1296.
strikeThru The strike through option to use.
textComposer The text composing engine to use.
tracking The amount of uniform spacing between multiple characters. Range: -1000 to 10000.
typename readonly The class name of the object.
underline Options for underlining the text.
useAutoLeading If true, uses the font’s built-in leading information.
verticalScale Character scaling (vertical) in proportion to horizontal scale. Range: 0 - 1000 as a percentage.
warpBend The warp bend percentage. Range: -100 to 100.
warpDirection The warp direction.
warpHorizontalDistortion The warp horizontal distortion percentage. Range: -100 to 100.
warpStyle The style of warp.
warpVerticalDistortion The warp vertical distortion percentage. Range: -100 to 100.
width The width of the bounding box for paragraph text.

Methods¶

convertToShape Converts the text object and its containing layer to a fill layer with the text changed to a clipping path.
createPath Creates a clipping path from the outlines of the actual text items (such as letters or words).
