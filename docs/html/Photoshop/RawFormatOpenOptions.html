RawFormatOpenOptions¶

Description¶

Options that can be specified when opening a document in RAW format.

Static Properties¶

bitsPerChannel The number of bits for each channel. Valid values: 8 or 16.
byteOrder The order in which bytes will be read. Valid only when ‘bits per channel’ = 16.
channelNumber The number of channels in the image. Valid only when ‘bits per channel’ = 16. Range: 1 to 56.
headerSize The number of bytes of information that will appear in the file before actual image information begins; that is, the number of zeroes inserted at the beginning of the file as placeholders. Range: 0 to 1919999.
height The image height (in pixels)
interleaveChannels If true, color values are stored sequentially.
retainHeader If true, the header is retained when saving.
width The image width (in pixels)
