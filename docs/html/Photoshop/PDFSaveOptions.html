PDFSaveOptions¶

Description¶

Options for saving a document in PDF format.

Static Properties¶

PDFCompatibility The PDF version to make the document compatible with.
PDFStandard The PDF standard to make the document compatible with.
alphaChannels If true, the alpha channels are saved.
annotations If true, the annotations are saved.
colorConversion If true, converts the color profile to the destination profile.
convertToEightBit If true, converts a 16-bit image to 8-bit for better compatibility with other applications.
description Description of the save options in use.
destinationProfile Describes the final RGB or CMYK output device, such as a monitor or press standard.
downSample The downsample method to use.
downSampleSize The size (in pixels per inch) to downsample images to if they exceed the value specified for ‘down sample size limit’.
downSampleSizeLimit Limits downsampling or subsampling to images that exceed this value (in pixels per inch).
downgradeColorProfile DEPRECATED, ( should the embedded color profile be downgraded to version 2 )
embedColorProfile If true, the color profile is embedded in the document.
embedFonts DEPRECATED. ( embed fonts? Only valid if a text layer is included )
embedThumbnail If true, includes a small preview image in Acrobat.
encoding The encoding method to use.
interpolation DEPRECATED. ( use image interpolation? )
jpegQuality The quality of the produced image. Valid only for JPEG-encoded PDF documents. Range: 0 to 12.
layers If true, the layers are saved.
optimizeForWeb If true, improves performance of PDFs on Web servers.
outputCondition An optional comment field for inserting descriptions of the output condition. The text is stored in the PDF/X file.
outputConditionID The identifier for the output condition.
preserveEditing If true, allows users to reopen the PDF in Photoshop with native Photoshop data intact.
presetFile The preset file to use for settings; overrides other settings.
profileInclusionPolicy If true, shows which profiles to include.
registryName The URL where the output condition is registered.
spotColors If true, the spot colors are saved.
tileSize The compression option. Valid only when encoding is JPEG2000.
transparency DEPRECATED.
useOutlines DEPRECATED. ( use outlines for text? Only valid if vector data is included )
vectorData DEPRECATED. ( include vector data )
view If true, opens the saved PDF in Acrobat.
