ScriptUIGraphics¶

Description¶

An object used to draw custom graphics, found in the graphics property of window, container, and control objects.

Allows a script to customize aspects of the element’s appearance, such as the color and font. Use an onDraw callback function to set these properties or call the functions.All measurements are in pixels.

Properties¶

backgroundColor The background color for containers; for non-containers, the parent background color.
currentPath readonly The current drawing path, encapsulated in a path object.
currentPoint readonly The current position in the current drawing path.
disabledBackgroundColor The background color for containers when disabled or inactive; for non-containers, the parent background color.
disabledForegroundColor The text color when the element is disabled or inactive.
font The default font to use for displaying text.
foregroundColor The text color.

Static Properties¶

BrushType readonly Contains the enumerated constants for the type argument of newBrush().
PenType readonly Contains the enumerated constants for the type argument of newPen().

Methods¶

closePath Closes the current path.
drawFocusRing Draws a focus ring within a region of this element.
drawImage Draws an image within a given region of the element.
drawOSControl Draw the platform-specific control associated with this element.
drawString Draw a string of text starting at a given point in this element, using a given drawing pen and font.
ellipsePath Defines an elliptical path within a given rectangular area in the currentPath object, which can be filled using fillPath() or stroked using strokePath().
fillPath Fills a path using a given painting brush.
lineTo Adds a path segment to the currentPath.
measureString Calculates the size needed to display a string using the given font.
moveTo Adds a given point to the currentPath, and makes it the current drawing position.
newBrush Creates a new painting brush object.
newPath Creates a new, empty path object.
newPen Creates a new drawing pen object.
rectPath Defines a rectangular path in the currentPath object.
strokePath Strokes the path segments of a path with a given drawing pen.
